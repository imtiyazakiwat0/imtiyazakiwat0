  1. RBI app for blinds

About:- Last year, in an RBI’s contest, I developed an android application using computer vision which can detect any Indian currency and speak out its denomination. The main motivation behind this app was to overcome the difficulties faced by visually impaired people to detect currency notes using the engravings on the notes. The app was designed in a way so that any blind person can use that app, using voice.

Drive link to APK:- RBIappforblinds.apk

Repo Link:- https://github.com/midopooler/RBI-app-for-blinds
